# AGV Basic Project

AGV(Automated Guided Vehicle) 기본 제어 프로젝트입니다.

## 📁 프로젝트 구조

```
1.agv_basic_proj/
├── agv_keyboard.py      # 키보드 제어 프로그램
├── Donkibot_i.py        # AGV 하드웨어 인터페이스
└── README.md
```

## 🚀 주요 기능

- **키보드 제어**: 키보드 입력을 통한 AGV 수동 제어
- **기본 모터 제어**: 전진, 후진, 좌회전, 우회전 등 기본 동작

## 💻 실행 방법

```bash
python agv_keyboard.py
```

## 📝 개발 환경

- Python 3.x
- Donkibot AGV 하드웨어
